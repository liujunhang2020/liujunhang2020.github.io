css实现瀑布流

本文最后更新于：6 天前

说明

在网页当中瀑布流是很常见的功能，一般来说都是通过js来实现瀑布流的功能。

但是如果功能较为简单的瀑布流，完全可以通过css来实现。

在本篇文章当中，将会通过两种方式来实现瀑布流的布局。

多列实现瀑布流

主要是通过下面的两个属性来实现瀑布流效果：

  • column-count: 设置列数
  • column-gap: 设置列间距

css代码如下:

body {
    margin: 4px;
    font-family: Helvetica;
}

.masonry {
    position: absolute;
    column-count: 4;
    column-gap: 0;
}

.item {
    padding: 2px;
    position: relative;
}

.item img {
    display: block;
    width: 100%;
    height: auto;
}

flex实现瀑布流

通过flex实现瀑布流效果，需要将主轴变为垂直方向，并且设置固定高度。

html代码和上面相同，下面是css代码：

body {
    margin: 4px;
    font-family: Helvetica;
}

.masonry {
    display: flex;
    flex-direction: column;
    flex-wrap: wrap;
    height: 3000px;
}

.item {
    position: relative;
    width: 25%;
    padding: 2px;
    box-sizing: border-box;
}

.item img {
    display: block;
    width: 100%;
    height: auto;
}
